# Activity Studio Contracts

This directory documents the public interface contracts for the Activity Studio module. These contracts define how other modules and the UI layer interact with the Activity domain.

## Contents

| Contract | File | Description |
|----------|------|-------------|
| Domain Interfaces | `domain-interfaces.md` | Repository and service interfaces for Activity, Relationship, Calendar, Bank entities |
| Application DTOs | `application-dtos.md` | Data transfer objects for cross-layer communication |
| UI Contracts | `ui-contracts.md` | ViewModel-to-View contracts and tab registration protocol |
| Persistence Contracts | `persistence-contracts.md` | EF Core entity configurations and DbContext integration points |
| Cross-Module Contracts | `cross-module-contracts.md` | How Activity Studio consumes WBS and Project modules |
